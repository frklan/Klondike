Help

If you access this app on iOS you can select the 'share' button in Safari and then 'add to homescreen' to create a stand alone shortcut on the homescreen

The following options are available from the menu

Track wallet

Makes the app fetch wallet specific data from the pool. Enter a zero lenght string to remove the tracking. Note: Do NOT enter your private address here!

Select pool

Set the pool from which the data shall be fetched. Missing any pool? Create a PR on github and it will be added if possible (e.g. having an API over HTTPS etc.)
